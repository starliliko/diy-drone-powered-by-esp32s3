# Mechanical — 机架设计

> DIY-Drone 项目的机械结构设计，包括 CAD 源文件、3D 打印件与图纸导出。

## 设计概述

当前机架面向 **更大轴距、无刷电机驱动** 的小型室内四旋翼平台，核心设计要点：

- 电机 / 桨叶 / 电池 / 电调的安装适配
- 飞控与传感器安装环境（减振、朝向）
- 重心布局与设备安装空间
- FDM 3D 打印可制造性与结构强度

## 目录结构

```text
mechanical/
├── 新版 4.5/       # 当前版本 — 新增机臂、桨保、上板等结构
│   ├── 完整体.iam  #   总装配体
│   ├── *.ipt       #   各零件（底板、顶板、机臂、桨保、起落架…）
│   └── OldVersions/
├── 旧完整版/       # 早期版本备份
│   ├── 完整体.iam
│   ├── 打印/       #   3MF 打印文件
│   └── OldVersions/
├── 打印/           # 最新打印文件（3MF / G-code）
└── PDF图/          # 导出图纸（PDF）
```

## 版本演进

| 版本 | 说明 |
|------|------|
| **旧完整版** | 初版整机方案，验证基本装配与飞行可行性 |
| **新版 4.5** | 当前迭代版本：新增独立机臂、桨叶保护罩、优化上板与底板布局 |

## 工具链

- **CAD**：Autodesk Inventor（`.ipt` 零件 / `.iam` 装配体）
- **切片 / 打印**：3MF 格式导出，FDM 打印
- **图纸导出**：PDF

## 机架组成（新版 4.5）

| 零件 | 文件 | 说明 |
|------|------|------|
| 底板 | `底板.ipt` | 主承力板，安装电机与电调 |
| 顶板 | `顶板.ipt` | 飞控与传感器安装板 |
| 上板 | `上板.ipt` | 顶部保护 / 附件安装 |
| 机臂 | `机臂.ipt` | 独立机臂结构 |
| 桨保 | `桨保.ipt` | 桨叶保护罩 |

| 铜柱 | `铜柱.ipt` | 板间连接件 |
