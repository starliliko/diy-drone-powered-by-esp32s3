## 2. 什么是频谱图

时域图回答的是：

- 波形随时间怎么变化。

频域图回答的是：

- 这个波形的能量主要集中在哪些频率。

例如：

- 机体慢慢晃，频率低。
- 电机振动，频率高。
- 纯随机抖动，会在比较宽的频段铺开。

所以频谱图不是“另一种画法”，而是把“抖动来自哪里”分解开看。

## 3. 为什么可以从频谱看出噪声

因为不同来源的信号，频率特征通常不同。

### 3.1 真实控制动态通常偏低频

对多旋翼来说，姿态、角速度、回正、慢摆、手持扰动，通常主要出现在较低频段。

常见表现：

- `0~几 Hz`：手持晃动、台架慢摆、慢漂
- `几 Hz ~ 十几 Hz`：姿态回正、外环动作、低频回摆

### 3.2 机械振动通常更高频

电机、桨叶、机架共振，常见在几十到上百 Hz。

常见表现：

- 某个很尖的峰：固定振动源
- 某一段很宽的包：结构共振或宽带振动

### 3.3 D 项最怕高频噪声

PID 里的 D 项本质上对变化率敏感，高频噪声一旦进来，D 会把它放大。

所以飞控里常见的做法是：

1. 先查机械振动。
2. 再做 gyro 低通、notch。
3. 最后才逐步加 D。

## 4. 频谱图里最常见的几种形态

### 4.1 低频大包

特征：

- 峰在 `0~10 Hz` 左右
- 宽、低、比较连续

更像：

- 手持扰动
- 台架慢摆
- 外环回正
- 低频 PID 摆动

不建议直接用 notch 去砍这部分。

### 4.2 高频窄峰

特征：

- 某个点附近突然很高
- 频率集中

更像：

- 电机机械频率
- 桨叶通过频率
- 某个结构共振点

这类最适合用 notch。

### 4.3 高频宽带抬升

特征：

- 一大片中高频都比底噪高

更像：

- 整体振动偏大
- 安装过硬
- 桨不平衡
- 结构杂振

这类不能只靠单个 notch，通常要先查机械。

## 5. 怎样判断“这个峰是不是噪声”

可以按下面 4 条来判断。

### 5.1 看频率位置

如果一段频率高到已经不像姿态真实变化，比如几十到上百 Hz，更像机械噪声。

### 5.2 看峰的形状

- 尖峰：固定振动源可能性大
- 宽包：随机噪声或结构共振可能性大

### 5.3 看工况变化

如果：

1. 电机不转时没有
2. 电机转起来就出来
3. 油门一变，峰强度或位置跟着变

那它基本就是机械振动相关噪声。

### 5.4 看滤波后的结果

如果对某个频点加 notch 后：

1. 该频点峰值明显下降
2. 时域波形变干净
3. 控制没有明显变钝

那说明你打中的大概率就是噪声。

## 6. 电机基频、桨叶通过频率、谐波

这是频谱分析里最容易混淆的一组概念。

### 6.1 电机机械基频

如果电机转速是 `RPM`，机械转频是：

`f_motor = RPM / 60`

例如：

- `6000 RPM -> 100 Hz`

### 6.2 桨叶通过频率

如果是双叶桨，每转一圈有 2 次叶片通过：

`f_blade = 叶片数 × RPM / 60`

例如双叶桨：

- `6000 RPM -> 200 Hz`

### 6.3 谐波

如果基频是 `100 Hz`，那：

- 二次谐波：`200 Hz`
- 三次谐波：`300 Hz`

频谱图里经常不是只看到一个峰，而是：

`f, 2f, 3f ...`

不过在当前项目的 VOFA 链路下，最常看到的是低频和中频的主峰，不一定能完整看到高阶谐波。

## 7. 采样率、奈奎斯特频率和分辨率

### 7.1 采样率

采样率越高，越能看到高频。

### 7.2 奈奎斯特频率

最大可观察频率约为：

`f_max = sample_rate / 2`

例如：

- `50 Hz` 采样，只能分析到 `25 Hz`
- `500 Hz` 采样，可以分析到 `250 Hz`
- `1000 Hz` 采样，可以分析到 `500 Hz`

这就是为什么 `50 Hz` 不够做电机振动频谱分析。

### 7.3 频率分辨率

粗略地说：

`df = sample_rate / N`

其中 `N` 是样本数。

例如：

- `500 Hz` 采样、记录 `10 秒`
- 样本数约 `5000`
- 分辨率约 `0.1 Hz`

记录越长，频率分辨率越高。

## 9. 怎样从时域数据得到频谱图

### 9.1 在 VOFA 里直接看

优点：

- 快
- 适合现场判断

缺点：

- 不够灵活
- 不方便做更严格的数值分析

### 9.2 导出 CSV 后分析

这是更标准的做法。

推荐两种算法：

1. FFT
2. Welch PSD

其中 Welch 更稳，更适合看噪声。

### 9.3 Python 示例

下面是一个最小可用示例，适合后续离线分析。

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from scipy.signal import welch

# 假设第5列是目标角速度数据
df = pd.read_csv("telemetry.csv", header=None)
y = df.iloc[:, 4].to_numpy(dtype=float)

fs = 500.0  # 采样率，按实际设置修改

f, pxx = welch(
    y,
    fs=fs,
    window="hann",
    nperseg=1024,
    noverlap=512,
    detrend="constant"
)

plt.figure(figsize=(10, 4))
plt.semilogy(f, pxx)
plt.xlabel("Frequency (Hz)")
plt.ylabel("PSD")
plt.title("Gyro Spectrum")
plt.grid(True)
plt.show()
```

## 10. 怎样根据频谱选 notch 参数

### 10.1 中心频率

把 notch 中心放在最强、最明确的那个机械峰上。

例如你前面的数据里，`103~109 Hz` 这簇峰最明显，所以中心频率选：

- `105 Hz`

### 10.2 带宽

带宽不要一开始就特别宽。

建议：

- 峰很尖：先窄一些
- 峰比较散：稍微宽一点

例如你前面的案例，用：

- `gyroNotchHz = 105`
- `gyroNotchBw = 15`

效果是合理的，因为它大致覆盖了这簇峰，而没有明显吃掉更宽的频段。

### 10.3 调参顺序

建议顺序：

1. 先定中心频率
2. 再调带宽
3. 再看时域是否变干净
4. 最后才回到 PID

## 11. 什么时候不该继续加滤波

如果频谱里：

1. 高频尖峰已经明显压下去了
2. 剩下主要是 `0~几 Hz` 的低频
3. 时域里主要是慢摆，不是高频发麻

那说明你已经不在“滤波阶段”，而是在“控制阶段”。

这时再继续加 notch，通常只会让系统更钝。

## 12. 频谱分析和 PID 的关系

可以把调试分成两个阶段。

### 阶段 A：先处理噪声

目标：

- 让高频机械振动不要污染陀螺仪
- 让 D 项不会把高频振动放大

主要动作：

- 查机械
- 调 LPF
- 调 notch

### 阶段 B：再调控制

目标：

- 解决低频摆动、回正无力、过冲、拖尾

主要动作：

- 调内环 P/I/D
- 调外环 P

简单说：

- 高频问题先看滤波
- 低频问题再看 PID

## 14. 常见误区

### 14.1 看到峰就打 notch

不对。

低频峰很多时候是控制动态，不该乱砍。

### 14.2 只靠滤波解决振动

不对。

如果桨不平衡、电机轴承不好、飞控安装太硬，滤波只能治标。

### 14.3 频谱好看了，就一定飞得好

不一定。

频谱主要帮助你定位噪声，不直接等于控制性能。

### 14.4 notch 越多越好

不对。

滤波器越多，相位延迟越大，控制会越来越钝。

## 15. 一套推荐工作流

建议每次都按下面走。

1. 电机不转，录静止频谱
2. 电机低油门，录静止频谱
3. 电机中油门，录静止频谱
4. 找明显的高频尖峰
5. 设置 notch
6. 再录一次同工况
7. 如果高频已明显下降，停止继续加滤波
8. 开始回到 PID 调参

## 17. 一句话总结

频谱图不是为了“让图好看”，而是为了回答两个问题：

1. 当前主要问题是不是高频机械噪声。
2. 如果是，应该在什么频率下刀。

当高频尖峰被压下之后，就不要继续沉迷滤波了，要回到真正决定飞行品质的 PID 和机械一致性上。
