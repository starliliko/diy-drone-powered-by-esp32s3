# 陀螺仪动态校准实现文档

## 实现日期
2026年1月14日

## 功能概述
参考 Flix 飞控项目的动态陀螺仪校准方法，在ESP-Drone项目中实现了**后台持续校准**功能，用于自适应补偿陀螺仪零点漂移。

### 核心理念
- **传统方法问题**：启动时1秒固定校准，要求静止，无法应对温漂
- **新方法优势**：着陆时持续后台校准，自动补偿零点变化，提升姿态保持性能

---

## 技术原理

### 一阶低通滤波器 (IIR)
使用最简单的指数移动平均（EMA）滤波器更新零点偏差：

```c
bias_new = bias_old + α * (gyro_raw - bias_old)
```

**参数说明：**
- `α = 0.001`：滤波系数（平滑因子）
- **时间常数 τ ≈ 1秒**（达到最终值63%的时间）
- 计算公式：`τ = Δt / α`，其中 `Δt = 1ms`（1kHz采样率）

### 工作流程
```
1. 检测飞行状态 → quadIsFlying (从 estimator_kalman 获取)
2. 着陆检测 → 记录着陆时间戳
3. 延迟2秒 → 避免着陆冲击干扰
4. 启动滤波 → 持续更新 gyroBias.x/y/z
5. 起飞时暂停 → 仅在静止时校准
```

## 代码实现

### 修改的文件

#### 1. `sensors_bmi088_spi_ms5611.c`
传感器HAL层，核心校准逻辑实现

**新增宏定义：**
```c
#define GYRO_DYNAMIC_CALIBRATION_ENABLED 1        // 使能动态校准
#define GYRO_DYNAMIC_CALIB_DELAY_MS M2T(2 * 1000) // 着陆后2秒延迟
#define GYRO_DYNAMIC_CALIB_ALPHA 0.001f           // 滤波系数
```

**新增全局变量：**
```c
static uint32_t landedTime = 0;                   // 着陆时间戳
static bool isLanded = false;                     // 着陆状态标志
static float gyroDynamicCalibAlpha = 0.001f;      // 可调滤波系数
static uint8_t gyroDynamicCalibEnabled = 1;       // 使能开关
```

**核心函数：**
```c
static void gyroDynamicCalibUpdate(int16_t gx, int16_t gy, int16_t gz, bool quadIsFlying)
{
    if (!gyroDynamicCalibEnabled) return;
    
    uint32_t currentTime = xTaskGetTickCount();
    
    // 着陆状态机
    if (!quadIsFlying && !isLanded) {
        landedTime = currentTime;  // 记录着陆时刻
        isLanded = true;
    } else if (quadIsFlying && isLanded) {
        isLanded = false;          // 起飞，重置状态
    }
    
    // 仅在着陆且超过延迟后更新
    if (isLanded && (currentTime - landedTime > GYRO_DYNAMIC_CALIB_DELAY_MS)) {
        // 一阶IIR滤波器更新
        gyroBias.x += gyroDynamicCalibAlpha * (gx - gyroBias.x);
        gyroBias.y += gyroDynamicCalibAlpha * (gy - gyroBias.y);
        gyroBias.z += gyroDynamicCalibAlpha * (gz - gyroBias.z);
    }
}
```

**集成到主循环：**
```c
// 在传感器读取循环中（1kHz）
if (gyroBiasFound) {
    processAccScale(accelRaw.x, accelRaw.y, accelRaw.z);
    #ifdef GYRO_DYNAMIC_CALIBRATION_ENABLED
    gyroDynamicCalibUpdate(gyroRaw.x, gyroRaw.y, gyroRaw.z, quadIsFlying);
    #endif
}
```

## 参数调优指南

### 滤波系数 α 的影响

| α 值   | 时间常数 | 收敛速度 | 噪声抑制 | 适用场景               |
| ------ | -------- | -------- | -------- | ---------------------- |
| 0.0001 | 10秒     | 很慢     | 极强     | 极稳定环境，长时间着陆 |
| 0.001  | 1秒      | 适中     | 强       | **推荐默认值**         |
| 0.01   | 0.1秒    | 快       | 中等     | 快速测试，温漂明显时   |
| 0.1    | 0.01秒   | 极快     | 弱       | 不推荐（过于敏感）     |

**计算公式：**
```
时间常数 τ = Δt / α
Δt = 1ms (1kHz采样率)

α = 0.001 → τ = 1ms / 0.001 = 1000ms = 1秒
```

### 延迟时间建议
- **2秒延迟**（默认）：适合大多数场景
- **更长延迟**：粗暴着陆或振动环境
- **更短延迟**：精确着陆，静止性能好

## 未来改进方向

### 1. 静止检测优化
当前仅依赖 `quadIsFlying` 判断，可能在桌面滑动时误判。建议：
```c
// 添加陀螺仪幅值检测
bool isStationary = (abs(gx) < THRESHOLD) && 
                    (abs(gy) < THRESHOLD) && 
                    (abs(gz) < THRESHOLD);
```

### 2. 自适应滤波系数
根据方差动态调整 α：
```c
// 静止时方差小 → 增大α加快收敛
// 运动时方差大 → 减小α避免误校准
float adaptiveAlpha = baseAlpha * (1.0f - min(variance / MAX_VAR, 1.0f));
```

### 3. 温度补偿联动
结合BMI088内部温度传感器：
```c
// 温度变化大时增大α
if (fabs(currentTemp - lastTemp) > TEMP_THRESHOLD) {
    alpha_temp_compensated = alpha * 2.0f;
}
```

### 4. 多状态校准
- 着陆校准（当前实现）
- 悬停微调（检测长时间悬停状态）
- 飞行中缓慢更新（极小α值）
