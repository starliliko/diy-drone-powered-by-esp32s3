# DIY-Drone

> 基于 ESP-Drone 二次开发的室内无刷四旋翼无人机，涵盖飞控固件、机架设计与地面站全链路。

本项目以 Espressif [ESP-Drone](https://github.com/espressif/esp-drone) 飞控工程为基础，将原有微型有刷方案重构为 **更大轴距、无刷电机驱动** 的四旋翼平台，并围绕该平台完成了飞控适配、传感器链路优化、安全机制完善、机架结构设计以及地面站调试闭环搭建。

## 项目亮点

| 方向 | 内容 |
|------|------|
| **飞控适配** | 基于 ESP32-S3 / ESP-IDF / FreeRTOS，对 ESP-Drone 飞控架构进行重构与平台适配 |
| **传感器链路** | 接入 BMI088（SPI）、气压计、光流，完成中断驱动采集、零偏校准、低通 / 陷波滤波 |
| **状态估计与控制** | 卡尔曼估计器、PID 级联控制、多飞行模式（姿态 / 定高 / 定点）、多控制源优先级切换 |
| **安全机制** | 遥控失联降级、控制源超时切换、解锁保护、异常紧急停机 |
| **机架设计** | Autodesk Inventor 建模 → 3D 打印打样 → 实机装配 → 结构迭代 |
| **地面站** | Node.js + WebSocket，实时遥测、3D 姿态可视化、远程控制、电机测试 |

## 系统架构

```text
┌──────────────┐  TCP/WiFi   ┌──────────────┐  WebSocket  ┌──────────────┐
│   ESP32-S3   │ ──────────▶ │  Node.js     │ ──────────▶ │   浏览器      │
│   飞控固件    │ ◀────────── │  地面站服务    │ ◀────────── │   Web UI     │
└──────────────┘   控制指令   └──────────────┘   操作指令   └──────────────┘
       │
       ├── BMI088（SPI）· 加速度 / 陀螺仪
       ├── 气压计（I2C）· 高度估计
       ├── 光流（UART）· 水平位置
       └── SBUS 遥控器 · 最高优先级控制源
```

### 飞控固件 — `software/firmware/`

- 传感器驱动与中断采集
- 卡尔曼状态估计、PID 级联控制
- 多飞行模式（姿态稳定 / 定高 / 定点）
- 多控制源优先级管理（SBUS > TCP > CRTP）
- 失联降级、解锁保护、紧急停机等安全链路

### 地面站 — `software/groundstation/`

- 遥测仪表盘与 Three.js 3D 姿态可视化
- 解锁 / 起飞 / 降落 / 紧急停止 等飞行控制
- 独立电机推力测试
- WebSocket 实时双向通信

### 机架 — `mechanical/`

- Autodesk Inventor 全参数化建模
- 电机 / 桨叶 / 电池 / 电调安装适配
- 重心布局、减振结构、强度优化
- 3D 打印打样与实物装配验证

## 技术栈

| 类别 | 技术 |
|------|------|
| MCU / 平台 | ESP32-S3 |
| 固件框架 | ESP-IDF 5.x · FreeRTOS |
| 传感器 | BMI088 · BMP388 · PMW3901 光流 |
| 通信接口 | SPI · I2C · UART · TCP · WiFi |
| 控制与估计 | PID 级联 · 扩展卡尔曼滤波 · 多模式状态管理 |
| 地面站 | Node.js · Express · WebSocket · Three.js |
| 机械设计 | Autodesk Inventor · FDM 3D 打印 |
