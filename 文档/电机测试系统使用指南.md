# 电机测试系统使用指南

## 概述

新的电机测试系统参考QGC地面站设计，提供了图形化界面来测试四个电机的输出性能。

## 系统架构

### 后端组件
- **motor_test.c/h**: 电机测试模块，提供参数和日志接口
- **system.c**: 在启动时初始化电机测试模块
- **LOG系统**: 导出电机实时输出数据到地面站

### 前端组件
- **分页导航**: 两个标签页（飞行监控 / 电机测试）
- **电机测试页**: 
  - 四个电机实时输出显示
  - 四个独立滑块控制各电机推力
  - 快捷测试按钮（顺序测试、全部测试、急停等）

## 功能说明

### 1. 电机卡片
每个电机卡片包含：
- **标题**: 显示电机编号和位置（如：M1 前右）
- **输出显示**: 实时显示当前输出百分比
- **滑块**: 拖动调节推力（0-100%）
- **数值显示**: 底部显示原始值（0-65535）

### 2. 测试控制按钮
- **顺序测试**: 自动依次测试每个电机（10%推力，持续1秒）
- **全部测试**: 同时给所有电机10%推力
- **急停**: 立即停止所有电机（紧急情况使用）
- **设置10%/20%**: 快速设置所有电机到指定推力

### 3. 参数系统
通过CRTP参数接口控制：
- `motortest.mode`: 测试模式（0=停止, 1=顺序, 2=单电机, 3=全部）
- `motortest.id`: 选择单个电机ID（0-3）
- `motortest.thrust`: 推力值（0-65535）
- `motortest.estop`: 急停标志

### 4. LOG系统
导出到地面站的实时数据：
- `motors.m1/m2/m3/m4`: 原始输出值（0-65535）
- `motors.m1_pct/m2_pct/m3_pct/m4_pct`: 百分比（0-100）

## 使用步骤

### 准备工作
1. ⚠️ **务必卸下螺旋桨**
2. 确保飞行器稳定固定（使用测试架或固定装置）
3. 检查电池电量充足
4. 编译并烧录固件到ESP32-S3

### 测试流程

#### 1. 启动系统
```bash
cd apptest
npm start
```

浏览器访问 `http://localhost:3000`

#### 2. 连接飞控
确保ESP-Drone已启动并连接到服务器，界面状态栏显示"已连接"

#### 3. 切换到电机测试页
点击顶部导航栏的 "⚙️ 电机测试" 标签

#### 4. 单个电机测试
- 拖动任意电机的滑块
- 观察电机响应和输出显示
- 建议从5-10%开始测试

#### 5. 顺序测试
- 点击"顺序测试"按钮
- 系统会自动依次测试M1→M2→M3→M4
- 每个电机运行1秒，间隔0.5秒

#### 6. 全部电机测试
- 点击"设置10%"或"设置20%"按钮
- 所有电机同时输出相同推力
- 观察飞行器是否平衡

#### 7. 紧急停止
- 任何时候点击"急停"按钮
- 所有电机立即停止输出

## API接口

### HTTP接口

#### 设置单个电机
```javascript
POST /api/motor/set
Content-Type: application/json

{
  "motorId": 0,      // 0-3 对应 M1-M4
  "thrust": 6554     // 0-65535 (10% ≈ 6554)
}
```

#### 顺序测试
```javascript
POST /api/motor/sequential
Content-Type: application/json

{
  "thrust": 6554     // 测试推力值
}
```

#### 急停
```javascript
POST /api/motor/estop
```

## 安全注意事项

⚠️ **重要安全提示**
1. **必须卸下螺旋桨再测试**
2. 从小推力（5-10%）开始逐步增加
3. 保持手指远离电机
4. 确保飞行器稳定固定
5. 发现异常立即按急停
6. 测试时保持电池监控，避免过放

## 故障排查

### 问题1：界面无法连接
- 检查ESP-Drone是否正常启动
- 检查服务器端口是否正确（默认8080）
- 查看控制台日志

### 问题2：滑块无响应
- 检查WebSocket连接状态
- 打开浏览器开发者工具查看网络请求
- 检查后端API是否正常

### 问题3：电机不转
- 检查电机是否正确连接
- 检查ESC是否校准
- 检查电池电压是否足够
- 查看固件日志输出

### 问题4：电机输出不一致
- 检查电机接线顺序
- 检查ESC参数设置
- 可能需要单独校准ESC
