## 1. 硬件概述

### 1.1 BMI088简介

BMI088是Bosch生产的高性能6轴惯性测量单元（IMU），专为无人机和机器人应用设计。

**核心特性**：
- **双芯片架构**：加速度计和陀螺仪是独立的两个芯片，各自有独立的SPI接口
- **加速度计**：±3g/±6g/±12g/±24g可选量程，最高1600Hz ODR
- **陀螺仪**：±125°/s ~ ±2000°/s可选量程，最高2000Hz ODR
- **温度传感器**：内置温度传感器
- **低噪声**：陀螺仪噪声密度 0.014°/s/√Hz

### 1.2 引脚定义

| 引脚  | 功能              | 说明         |
| ----- | ----------------- | ------------ |
| VDD   | 电源              | 1.71V - 3.6V |
| VDDIO | IO电源            | 1.2V - 3.6V  |
| GND   | 地                |              |
| CSB1  | 加速度计片选      | 低电平有效   |
| CSB2  | 陀螺仪片选        | 低电平有效   |
| SCK   | SPI时钟           | 最高10MHz    |
| SDI   | SPI数据输入(MOSI) |              |
| SDO   | SPI数据输出(MISO) |              |
| INT1  | 加速度计中断1     | 可配置极性   |
| INT2  | 加速度计中断2     | 可配置极性   |
| INT3  | 陀螺仪中断        | 可配置极性   |
| INT4  | 陀螺仪中断        | 可配置极性   |

### 1.3 硬件连接示例（ESP32-S3）

```
ESP32-S3          BMI088
---------         ------
GPIO11 (MOSI) --> SDI
GPIO13 (MISO) <-- SDO
GPIO12 (SCK)  --> SCK
GPIO10        --> CSB1 (ACC)
GPIO16        --> CSB2 (GYRO)
GPIO4         <-- INT1 (ACC DRDY)
GPIO5         <-- INT3 (GYRO DRDY)
```

## 2. SPI通信协议

### 2.1 基本协议

BMI088使用SPI Mode 3（CPOL=1, CPHA=1），最大时钟频率10MHz。

**读操作**：
```
主机发送: [地址 | 0x80] [dummy] [dummy...]
从机返回: [无效]        [无效]  [数据...]
```

**写操作**：
```
主机发送: [地址 | 0x00] [数据]
从机返回: [无效]        [无效]
```

### 2.2 加速度计SPI特殊要求

> ⚠️ **关键差异**：加速度计读取时需要额外的dummy byte！

**加速度计读取时序**：
```
主机: [地址|0x80] [dummy] [dummy] [dummy...]
从机: [无效]      [无效]  [数据]  [数据...]
                    ↑
                  额外dummy byte（仅加速度计需要）
```

**陀螺仪读取时序**（标准SPI）：
```
主机: [地址|0x80] [dummy] [dummy...]
从机: [无效]      [数据]  [数据...]
```

### 2.3 SPI读写函数实现

```c
// 加速度计读取（需要额外dummy byte）
bool bmi088_acc_read_reg(bmi088_dev_t *dev, uint8_t reg_addr, uint8_t *data, size_t length)
{
    // 发送: 地址 + dummy + 数据区
    uint8_t tx_buf[length + 2];
    uint8_t rx_buf[length + 2];
    
    tx_buf[0] = reg_addr | 0x80;  // 读标志
    memset(&tx_buf[1], 0x00, length + 1);
    
    spi_transfer(tx_buf, rx_buf, length + 2);
    
    // 数据从第3个字节开始（跳过2个dummy）
    memcpy(data, &rx_buf[2], length);
    return true;
}

// 陀螺仪读取（标准SPI）
bool bmi088_gyro_read_reg(bmi088_dev_t *dev, uint8_t reg_addr, uint8_t *data, size_t length)
{
    uint8_t tx_buf[length + 1];
    uint8_t rx_buf[length + 1];
    
    tx_buf[0] = reg_addr | 0x80;
    memset(&tx_buf[1], 0x00, length);
    
    spi_transfer(tx_buf, rx_buf, length + 1);
    
    // 数据从第2个字节开始
    memcpy(data, &rx_buf[1], length);
    return true;
}

// 写寄存器（加速度计和陀螺仪相同）
bool bmi088_write_reg(spi_device *dev, uint8_t reg_addr, uint8_t value)
{
    uint8_t tx_buf[2] = {reg_addr & 0x7F, value};  // 写标志 = 0
    return spi_write(dev, tx_buf, 2);
}
```

## 3. 寄存器详解

### 3.1 加速度计寄存器

| 地址      | 名称             | 读/写 | 默认值 | 说明                       |
| --------- | ---------------- | ----- | ------ | -------------------------- |
| 0x00      | ACC_CHIP_ID      | R     | 0x1E   | 芯片ID                     |
| 0x02      | ACC_ERR_REG      | R     | 0x00   | 错误状态                   |
| 0x03      | ACC_STATUS       | R     | -      | 状态（bit7=数据就绪）      |
| 0x12-0x17 | ACC_DATA         | R     | -      | 加速度数据（X/Y/Z各2字节） |
| 0x1D      | ACC_INT_STAT_1   | R     | -      | 中断状态（bit7=数据就绪）  |
| 0x22-0x23 | ACC_TEMP         | R     | -      | 温度数据                   |
| **0x40**  | **ACC_CONF**     | R/W   | 0x80   | **采样配置（关键！）**     |
| 0x41      | ACC_RANGE        | R/W   | 0x01   | 量程配置                   |
| 0x53      | ACC_INT1_IO_CONF | R/W   | 0x00   | INT1引脚配置               |
| 0x58      | ACC_INT_MAP_DATA | R/W   | 0x00   | 中断映射                   |
| 0x7C      | ACC_PWR_CONF     | R/W   | 0x03   | 电源配置（0=Active）       |
| 0x7D      | ACC_PWR_CTRL     | R/W   | 0x00   | 电源控制（0x04=使能）      |
| 0x7E      | ACC_SOFTRESET    | W     | -      | 写0xB6触发软复位           |

### 3.2 ACC_CONF 寄存器详解（0x40）

> ⚠️ **这是导致加速度计不工作的最常见问题！**

```
Bit 7:   acc_bwp[3]  - 必须设为1！否则加速度计不采样！
Bit 6:4: acc_bwp[2:0] - 带宽设置
Bit 3:0: acc_odr      - 输出数据率
```

**正确配置值**：

| 配置 | 值               | 说明                      |
| ---- | ---------------- | ------------------------- |
| 0xAC | 1600Hz, OSR4带宽 | **推荐（Crazyflie使用）** |
| 0x8C | 1600Hz, OSR4带宽 | 等效                      |
| 0xAB | 800Hz, OSR4带宽  |                           |
| 0xA8 | 100Hz, OSR4带宽  |                           |

❌ **错误值**：0x2C（bit7=0，加速度计不会正常采样！）

### 3.3 ACC_ERR_REG 寄存器（0x02）

用于诊断配置错误：

```
Bit 0:   fatal_err   - 致命错误
Bit 2-4: error_code  - 错误代码
         0 = 无错误
         1 = 配置错误（最常见）
         2 = 错误的寄存器访问
```

### 3.4 陀螺仪寄存器

| 地址      | 名称                   | 读/写 | 默认值 | 说明             |
| --------- | ---------------------- | ----- | ------ | ---------------- |
| 0x00      | GYRO_CHIP_ID           | R     | 0x0F   | 芯片ID           |
| 0x02-0x07 | GYRO_DATA              | R     | -      | 角速度数据       |
| 0x0A      | GYRO_INT_STAT_1        | R     | -      | 中断状态         |
| 0x0F      | GYRO_RANGE             | R/W   | 0x00   | 量程配置         |
| 0x10      | GYRO_BANDWIDTH         | R/W   | 0x00   | 带宽/ODR配置     |
| 0x11      | GYRO_LPM1              | R/W   | 0x00   | 电源模式         |
| 0x14      | GYRO_SOFTRESET         | W     | -      | 写0xB6触发软复位 |
| 0x15      | GYRO_INT_CTRL          | R/W   | 0x00   | 中断控制         |
| 0x16      | GYRO_INT3_INT4_IO_CONF | R/W   | 0x00   | INT3/4配置       |
| 0x18      | GYRO_INT3_INT4_IO_MAP  | R/W   | 0x00   | 中断映射         |

### 4.2 初始化代码实现

```c
bool bmi088_init(bmi088_dev_t *dev)
{
    // ==================== 步骤1: 激活SPI模式 ====================
    // BMI088加速度计上电后默认处于I2C模式
    // 必须通过SPI读取来切换到SPI模式
    uint8_t dummy;
    bmi088_acc_read_reg(dev, 0x00, &dummy, 1);  // 第一次读取(无效数据)
    delay_ms(2);
    bmi088_acc_read_reg(dev, 0x00, &dummy, 1);  // 第二次读取(有效数据)
    delay_ms(2);
    
    if (dummy != 0x1E) {
        printf("ACC SPI mode activation failed! chip_id=0x%02X\n", dummy);
        return false;
    }
    
    // ==================== 步骤2: 软复位 ====================
    uint8_t reset_cmd = 0xB6;
    bmi088_acc_write_reg(dev, 0x7E, &reset_cmd, 1);
    delay_ms(1);
    
    // 复位后必须重新激活SPI模式!
    bmi088_acc_read_reg(dev, 0x00, &dummy, 1);
    delay_ms(2);
    bmi088_acc_read_reg(dev, 0x00, &dummy, 1);
    delay_ms(2);
    
    // ==================== 步骤3: 配置电源 ====================
    // 【关键顺序】先退出Suspend，再使能加速度计
    
    // 3.1 退出Suspend模式
    uint8_t pwr_conf = 0x00;
    bmi088_acc_write_reg(dev, 0x7C, &pwr_conf, 1);  // PWR_CONF = 0x00
    delay_ms(1);  // 等待450us以上
    
    // 验证写入
    uint8_t pwr_conf_read;
    bmi088_acc_read_reg(dev, 0x7C, &pwr_conf_read, 1);
    if (pwr_conf_read != 0x00) {
        printf("PWR_CONF write failed! read=0x%02X\n", pwr_conf_read);
        return false;
    }
    
    // 3.2 使能加速度计
    uint8_t pwr_ctrl = 0x04;
    bmi088_acc_write_reg(dev, 0x7D, &pwr_ctrl, 1);  // PWR_CTRL = 0x04
    
    // 【关键！】必须等待50ms让模拟电路稳定
    delay_ms(50);
    
    // ==================== 步骤4: 配置采样参数 ====================
    // 【核心修复】ACC_CONF的bit7必须设为1!
    // 0xAC = 0b10101100 = bit7(1) + bwp(010) + odr(1100)
    //                           ↑ 这个bit必须为1!
    uint8_t acc_conf = 0xAC;  // 1600Hz ODR, OSR4带宽
    bmi088_acc_write_reg(dev, 0x40, &acc_conf, 1);
    delay_ms(10);
    
    // 配置量程
    uint8_t acc_range = 0x01;  // 6G
    bmi088_acc_write_reg(dev, 0x41, &acc_range, 1);
    delay_ms(10);
    
    // ==================== 步骤5: 验证配置 ====================
    uint8_t err_reg, status_reg, conf_read;
    bmi088_acc_read_reg(dev, 0x02, &err_reg, 1);
    bmi088_acc_read_reg(dev, 0x03, &status_reg, 1);
    bmi088_acc_read_reg(dev, 0x40, &conf_read, 1);
    
    printf("ACC_ERR=0x%02X, STATUS=0x%02X, CONF=0x%02X\n", 
           err_reg, status_reg, conf_read);
    
    if (err_reg != 0x00) {
        printf("ACC configuration error! error_code=%d\n", (err_reg >> 2) & 0x07);
        return false;
    }
    
    // ==================== 步骤6: 初始化陀螺仪 ====================
    // 陀螺仪初始化相对简单
    uint8_t gyro_reset = 0xB6;
    bmi088_gyro_write_reg(dev, 0x14, &gyro_reset, 1);
    delay_ms(80);  // 陀螺仪需要30ms以上启动时间
    
    // 配置陀螺仪
    uint8_t gyro_range = 0x01;  // 1000 dps
    uint8_t gyro_bw = 0x02;     // 1000Hz ODR, 116Hz带宽
    bmi088_gyro_write_reg(dev, 0x0F, &gyro_range, 1);
    delay_ms(2);
    bmi088_gyro_write_reg(dev, 0x10, &gyro_bw, 1);
    delay_ms(2);
    
    return true;
}
```

## 5. 数据读取

### 5.1 加速度计数据格式

数据存储在0x12-0x17，每轴2字节（LSB + MSB），16位有符号整数：

```c
bool bmi088_read_acc(bmi088_dev_t *dev, int16_t *ax, int16_t *ay, int16_t *az)
{
    uint8_t data[6];
    
    // 一次读取6字节
    if (!bmi088_acc_read_reg(dev, 0x12, data, 6)) {
        return false;
    }
    
    // 组装16位数据（小端序）
    *ax = (int16_t)((data[1] << 8) | data[0]);
    *ay = (int16_t)((data[3] << 8) | data[2]);
    *az = (int16_t)((data[5] << 8) | data[4]);
    
    return true;
}
```

### 5.2 陀螺仪数据格式

数据存储在0x02-0x07，格式相同：

```c
bool bmi088_read_gyro(bmi088_dev_t *dev, int16_t *gx, int16_t *gy, int16_t *gz)
{
    uint8_t data[6];
    
    if (!bmi088_gyro_read_reg(dev, 0x02, data, 6)) {
        return false;
    }
    
    *gx = (int16_t)((data[1] << 8) | data[0]);
    *gy = (int16_t)((data[3] << 8) | data[2]);
    *gz = (int16_t)((data[5] << 8) | data[4]);
    
    return true;
}
```

### 5.3 数据转换为物理单位

```c
// 量程比例因子表
static const float acc_scale[] = {
    3.0f / 32768.0f,   // ±3g  -> g/LSB
    6.0f / 32768.0f,   // ±6g
    12.0f / 32768.0f,  // ±12g
    24.0f / 32768.0f   // ±24g
};

static const float gyro_scale[] = {
    2000.0f / 32768.0f,  // ±2000 dps -> dps/LSB
    1000.0f / 32768.0f,  // ±1000 dps
    500.0f / 32768.0f,   // ±500 dps
    250.0f / 32768.0f,   // ±250 dps
    125.0f / 32768.0f    // ±125 dps
};

// 转换为物理单位
float acc_x_g = raw_ax * acc_scale[range];       // 单位: g
float gyro_x_dps = raw_gx * gyro_scale[range];   // 单位: °/s
float gyro_x_rad = gyro_x_dps * M_PI / 180.0f;   // 单位: rad/s
```

### 5.4 温度读取

```c
bool bmi088_read_temperature(bmi088_dev_t *dev, float *temp_c)
{
    uint8_t data[2];
    bmi088_acc_read_reg(dev, 0x22, data, 2);
    
    // 11位有符号数据
    int16_t temp_raw = (int16_t)((data[1] << 8) | data[0]);
    temp_raw = temp_raw >> 5;
    
    if (temp_raw > 1023) {
        temp_raw -= 2048;  // 补码转换
    }
    
    *temp_c = temp_raw * 0.125f + 23.0f;
    return true;
}
```

## 6. 中断配置

### 6.1 加速度计中断（INT1）

```c
bool bmi088_configure_acc_interrupt(bmi088_dev_t *dev)
{
    // 配置INT1引脚
    // Bit 3: 输出使能 (1=使能)
    // Bit 2: 输出模式 (0=推挽, 1=开漏)
    // Bit 1: 极性 (0=低电平有效, 1=高电平有效)
    uint8_t int1_conf = 0x08;  // 输出使能, 推挽, 低电平有效
    bmi088_acc_write_reg(dev, 0x53, &int1_conf, 1);
    delay_ms(1);
    
    // 映射数据就绪中断到INT1
    // Bit 2: DRDY -> INT1
    uint8_t int_map = 0x04;
    bmi088_acc_write_reg(dev, 0x58, &int_map, 1);
    delay_ms(1);
    
    return true;
}
```

### 6.2 陀螺仪中断（INT3）

```c
bool bmi088_configure_gyro_interrupt(bmi088_dev_t *dev)
{
    // 配置INT3引脚
    // Bit 0: 极性 (0=低电平有效, 1=高电平有效)
    // Bit 1: 输出模式 (0=推挽)
    uint8_t int3_conf = 0x01;  // 高电平有效, 推挽
    bmi088_gyro_write_reg(dev, 0x16, &int3_conf, 1);
    delay_ms(1);
    
    // 使能数据就绪中断
    uint8_t int_ctrl = 0x80;
    bmi088_gyro_write_reg(dev, 0x15, &int_ctrl, 1);
    delay_ms(1);
    
    // 映射到INT3
    uint8_t int_map = 0x01;
    bmi088_gyro_write_reg(dev, 0x18, &int_map, 1);
    delay_ms(1);
    
    return true;
}
```

## 7. 调试经验与常见问题

### 7.1 问题1：加速度计数据不更新（最常见！）

**现象**：
- 加速度计只读到一次数据，之后数据不变
- 陀螺仪数据正常更新
- `INT_STAT_1` 始终为 0x00

**诊断**：
```c
uint8_t err_reg, status_reg, conf_reg;
bmi088_acc_read_reg(dev, 0x02, &err_reg, 1);
bmi088_acc_read_reg(dev, 0x03, &status_reg, 1);
bmi088_acc_read_reg(dev, 0x40, &conf_reg, 1);
printf("ERR=0x%02X, STATUS=0x%02X, CONF=0x%02X\n", err_reg, status_reg, conf_reg);
```

如果看到 `ERR=0x04`（error_code=1），说明配置错误！

**根本原因**：`ACC_CONF` 寄存器的 **bit7 必须设为1**！

```c
// ❌ 错误
uint8_t conf = (bwp << 4) | odr;  // = 0x2C (bit7=0)

// ✅ 正确
uint8_t conf = 0x80 | (bwp << 4) | odr;  // = 0xAC (bit7=1)
```

### 7.2 问题2：SPI通信失败

**现象**：读取芯片ID返回0x00或0xFF

**检查项**：
1. SPI模式是否正确（Mode 3: CPOL=1, CPHA=1）
2. CS引脚是否正确控制
3. 加速度计第一次读取必须丢弃（dummy read）

```c
// 加速度计首次通信必须执行dummy read
uint8_t dummy;
bmi088_acc_read_reg(dev, 0x00, &dummy, 1);  // 返回无效
delay_ms(2);
bmi088_acc_read_reg(dev, 0x00, &dummy, 1);  // 返回有效
```

### 7.3 问题3：软复位后加速度计无响应

**原因**：软复位后加速度计回到I2C模式，必须重新激活SPI模式。

```c
// 软复位
bmi088_acc_write_reg(dev, 0x7E, 0xB6);
delay_ms(1);

// 【必须】重新激活SPI模式
bmi088_acc_read_reg(dev, 0x00, &dummy, 1);
delay_ms(2);
bmi088_acc_read_reg(dev, 0x00, &dummy, 1);
```

### 7.4 问题4：PWR_CTRL写入后数据仍不更新

**原因**：PWR_CTRL写入后必须等待50ms让模拟电路稳定。

```c
bmi088_acc_write_reg(dev, 0x7D, 0x04);
delay_ms(50);  // ← 关键！不能省略
```

| 检查项          | 期望值 | 如何验证                     |
| --------------- | ------ | ---------------------------- |
| ACC芯片ID       | 0x1E   | 读取0x00寄存器               |
| GYRO芯片ID      | 0x0F   | 读取0x00寄存器               |

### 7.6 诊断代码

```c
void bmi088_diagnose(bmi088_dev_t *dev)
{
    uint8_t chip_id, err_reg, status, pwr_conf, pwr_ctrl, acc_conf, int_stat;
    
    bmi088_acc_read_reg(dev, 0x00, &chip_id, 1);
    bmi088_acc_read_reg(dev, 0x02, &err_reg, 1);
    bmi088_acc_read_reg(dev, 0x03, &status, 1);
    bmi088_acc_read_reg(dev, 0x7C, &pwr_conf, 1);
    bmi088_acc_read_reg(dev, 0x7D, &pwr_ctrl, 1);
    bmi088_acc_read_reg(dev, 0x40, &acc_conf, 1);
    bmi088_acc_read_reg(dev, 0x1D, &int_stat, 1);
    
    printf("========== BMI088 Diagnostic ==========\n");
    printf("CHIP_ID:   0x%02X (expect 0x1E) %s\n", chip_id, chip_id==0x1E?"OK":"FAIL");
    printf("ERR_REG:   0x%02X (expect 0x00) %s\n", err_reg, err_reg==0x00?"OK":"ERROR");
    printf("STATUS:    0x%02X (bit7=%d=drdy)\n", status, (status>>7)&1);
    printf("PWR_CONF:  0x%02X (expect 0x00) %s\n", pwr_conf, pwr_conf==0x00?"OK":"SUSPEND");
    printf("PWR_CTRL:  0x%02X (expect 0x04) %s\n", pwr_ctrl, pwr_ctrl==0x04?"OK":"DISABLED");
    printf("ACC_CONF:  0x%02X (bit7=%d) %s\n", acc_conf, (acc_conf>>7)&1, 
           (acc_conf&0x80)?"OK":"CRITICAL-bit7=0!");
    printf("INT_STAT:  0x%02X (bit7=%d=drdy)\n", int_stat, (int_stat>>7)&1);
    
    if ((acc_conf & 0x80) == 0) {
        printf("\n*** CRITICAL: ACC_CONF bit7=0! Accelerometer will NOT sample! ***\n");
        printf("*** Fix: Set ACC_CONF = 0x%02X instead of 0x%02X ***\n", 
               acc_conf|0x80, acc_conf);
    }
}
```

### 8.1 头文件（bmi088.h）

```c
#ifndef BMI088_H
#define BMI088_H

#include <stdint.h>
#include <stdbool.h>

// 寄存器地址
#define BMI088_ACC_CHIP_ID      0x00
#define BMI088_ACC_ERR_REG      0x02
#define BMI088_ACC_STATUS       0x03
#define BMI088_ACC_DATA         0x12
#define BMI088_ACC_INT_STAT_1   0x1D
#define BMI088_ACC_CONF         0x40
#define BMI088_ACC_RANGE        0x41
#define BMI088_ACC_INT1_IO_CONF 0x53
#define BMI088_ACC_INT_MAP      0x58
#define BMI088_ACC_PWR_CONF     0x7C
#define BMI088_ACC_PWR_CTRL     0x7D
#define BMI088_ACC_SOFTRESET    0x7E

#define BMI088_GYRO_CHIP_ID     0x00
#define BMI088_GYRO_DATA        0x02
#define BMI088_GYRO_RANGE       0x0F
#define BMI088_GYRO_BANDWIDTH   0x10
#define BMI088_GYRO_SOFTRESET   0x14
#define BMI088_GYRO_INT_CTRL    0x15
#define BMI088_GYRO_INT3_CONF   0x16
#define BMI088_GYRO_INT_MAP     0x18

// 配置值
#define BMI088_ACC_CONF_1600HZ  0xAC  // 1600Hz, OSR4 (bit7=1!)
#define BMI088_ACC_RANGE_6G     0x01
#define BMI088_GYRO_RANGE_1000  0x01
#define BMI088_GYRO_BW_1000HZ   0x02

typedef struct {
    void *acc_spi;
    void *gyro_spi;
    bool initialized;
} bmi088_t;

typedef struct {
    int16_t x, y, z;
} bmi088_data_t;

bool bmi088_init(bmi088_t *dev);
bool bmi088_read_acc(bmi088_t *dev, bmi088_data_t *data);
bool bmi088_read_gyro(bmi088_t *dev, bmi088_data_t *data);
void bmi088_diagnose(bmi088_t *dev);

#endif
```

### 8.2 使用示例

```c
#include "bmi088.h"

void app_main(void)
{
    bmi088_t imu;
    
    // 初始化SPI总线...
    
    // 初始化BMI088
    if (!bmi088_init(&imu)) {
        printf("BMI088 init failed!\n");
        bmi088_diagnose(&imu);  // 打印诊断信息
        return;
    }
    
    printf("BMI088 initialized successfully!\n");
    
    // 主循环
    while (1) {
        bmi088_data_t acc, gyro;
        
        if (bmi088_read_acc(&imu, &acc) && bmi088_read_gyro(&imu, &gyro)) {
            printf("ACC: X=%d Y=%d Z=%d | GYRO: X=%d Y=%d Z=%d\n",
                   acc.x, acc.y, acc.z, gyro.x, gyro.y, gyro.z);
        }
        
        vTaskDelay(pdMS_TO_TICKS(10));  // 100Hz
    }
}
```
